# QA Checklists

В репозитории представлены чек-листы для тестирования веб- и мобильных приложений.

Документация включает:
- функциональные проверки
- исследовательское тестирование
- проверки жизненного цикла приложения
- проверки на прерывания и мобильную специфику

Чек-листы оформлены в табличном формате и могут использоваться
для повторного и регрессионного тестирования.
